# Paginação
Projeto para estudos de paginação em ASP.NET Core

## Usando

- .Net Core 6
- Entity Framework 6
- Bogus
- AutoBogus
- Sqlite

## Instalar os pacotes

- Bogus
- AutoBogus
- Microsoft.EntityFrameworkCore.Design
- Microsoft.EntityFrameworkCore.Sqlite
- Microsoft.EntityFrameworkCore.Tools

## Executar o comando

```dotnet ef database update```
